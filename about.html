Technologies Used

These are the three foundational technologies we used to create DeskSpacing.com

Bootstrap was used in the creation of the user interface of DeskSpacing.com , including the landing page.

ThreeJS

ThreeJS made up the majority of the code for DeskSpacing.com , it allowed us to load in 3D models and manipulate them in the browser.

blender

Blender was used to create all 3D models for DeskSpacing.com.
